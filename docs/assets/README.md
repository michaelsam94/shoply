# Media Placeholders

Drop screenshots/gifs here and reference them from the main README.

Suggested files:

- `home.png`
- `product-detail.png`
- `cart.png`
- `checkout.gif`
- `auth-login.png`
- `profile.png`

Tips:

- Use PNG for static screens.
- Use short GIF/MP4 clips (5-10s) for checkout flow demos.
- Prefer 1080px width captures for clean LinkedIn/GitHub previews.
